# Formula 1 Strategy & Performance Analytics Dashboard

## Project Overview

This project analyzes Formula 1 race results, driver performance, constructor competitiveness, and race strategy using Python and Power BI.

The goal is to generate actionable insights for team principals, race strategists, and performance analysts.

---

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Power BI
- Git
- GitHub

---

## Business Questions

- Which drivers consistently outperform expectations?
- Which teams maximize championship points?
- How strongly does qualifying affect race outcomes?
- Which circuits provide the greatest overtaking opportunities?
